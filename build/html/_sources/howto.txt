第三步：添加节点
~~~~~~~~~~~~~~~~

1.输入要扫描的域名

2.若没有安装python，请先安装python依赖于Python 2.7.* 

3.执行命令

-  ::

    python -c "exec(__import__('urllib').urlopen('http://bugscan.net/xxxxxxxxxxxxxxx').read())" -m 50

4.根据需要调整选项

5.点击开始扫描按钮
